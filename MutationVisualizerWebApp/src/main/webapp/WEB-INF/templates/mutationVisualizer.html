Create your protein

Welcome to the web application to create your own animal! Use the DNA spiral and change the sequence to modify the animal. Changes you make will be represented in a picture on this site. Modify and learn about the wonders of DNA mutation!

Tutorial section

Step 1

Modify the sequence

Change the pins in the DNA spiral model plugged into the computer to change the sequence. Simply fill the spiral and make sure every pin is in the spiral.

Step 2

Press the button!

Once you've created a sequence you can scroll down to the GENERATE button to generate the changes in your protein.

Step 3

Learn!

See what changes have happened in your protein and learn about them. The changes will be shown and the text box will describe the effects of the changes.

Visualization

Now it's time to create the animal. Make sure the DNA spiral is setup right and click the generate button under the picture. The picture will change and the text box will describe your changes.
